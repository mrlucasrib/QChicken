## :information_source: How To Use

### Prerequisites

- C++ Compiler (gcc, clang)
- Qt Framework (qmake and libs)

### Usage

From your command line:
```sh
# Clone repository
$ git clone https://github.com/mrlucasrib/QChicken.git
$ cd QChicken
$ qmake QChicken.pro
$ make
$ sudo chmod +x QChicken
$ ./QChicken
```
